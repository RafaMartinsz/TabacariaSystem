# Tabacaria Hub

Sistema web desenvolvido em ASP.NET Core MVC para gerenciamento de produtos, clientes e vendas.

## Funcionalidades

* Login de usuários
* Cadastro de clientes
* Cadastro de produtos
* Controle de vendas
* Exclusão e edição de registros
* Interface personalizada

## Tecnologias Utilizadas

* C#
* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* Bootstrap

## Objetivo

Desenvolver uma aplicação para auxiliar no controle administrativo e operacional de uma tabacaria, permitindo maior organização das informações e agilidade nos processos internos.
